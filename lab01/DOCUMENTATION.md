### Порождающий шаблон проектирования — строитель (Builder): C#

**Класс `Pizza`** — это тип данных для объектов со строковыми полями размером (`Size`) и типом корочки (`Crust`), а также логическими: колбасой (`Sausage`), сыром (`Cheese`), грибами (`Mushrooms`) и оливками (`Olives`); имеет конструктор для всех этих параметров.

**Класс `PizzaBuilder`** реализует рассматриваемый паттерн, устанавливая параметры через методы (`size(string)`, `crust(string)`, `sausage(bool)`, `cheese(bool)`, `mushrooms(bool)` и `olives(bool)`; каждый из них возвращает текущий объект, образуя цепочку вызовов) и для создания/возвращения объекта `Pizza` с ними — `build()`.

**Интерфейс `Windows.Forms`** содержит виджеты для выбора параметров: два `comboBox` (размер пиццы и тип корочки) и четыре `checkBox` (булевы); кнопки `button1`, создающую объект напрямую через конструктор, и `button2` — с использованием шаблона; `richTextBox1`, отображающий результаты.

**Вывод**: паттерн-строитель пошагово конструирует сложные объекты (со многими опциональными параметрами) другого класса независимо от него, ради удобства их настройки.
