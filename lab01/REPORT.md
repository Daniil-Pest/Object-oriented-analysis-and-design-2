### Шаблон проектирования — строитель (Builder): C#

**Класс `Pizza`** — это тип данных для объектов со строковыми полями размером (`Size`) и типом корочки (`Crust`), а также булевыми: колбасой (`Sausage`), сыром (`Cheese`), грибами (`Mushrooms`) и оливками (`Olives`); имеет конструктор для всех этих параметров.

**Класс `PizzaBuilder`** реализует рассматриваемый паттерн, работая через методы для установки параметров (`size`, `crust`, `sausage`, `cheese`, `mushrooms` и `olives`; каждый из них возвращает текущий объект `PizzaBuilder`, образуя цепочку вызовов) и для создания/возвращения объекта `Pizza` с ними — `build()`.

**Интерфейс `Windows Forms`** содержит виджеты для выбора параметров: `ComboBox` (размер пиццы и тип корочки) и четыре `CheckBox`; кнопки `button1`, создающую объект `Pizza` напрямую через конструктор, и `button2` — с использованием `PizzaBuilder`; `richTextBox1`, отображающий результаты.

**Вывод**: шаблон-строитель создаёт объекты другого класса независимо от него, ради удобства его настройки.
